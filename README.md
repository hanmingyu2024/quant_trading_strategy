# Quantitative Trading Strategy System

量化交易策略系统

## 项目简介

这是一个基于Python的量化交易策略系统,支持多种交易策略的回测和实盘交易。系统采用前后端分离架构,提供友好的Web界面进行策略开发、回测和监控。

## 主要功能

- 策略开发：支持编写自定义交易策略
- 数据获取：对接多个数据源获取历史和实时行情数据
- 回测系统：提供策略回测功能,支持参数优化
- 实盘交易：支持接入多个交易所进行实盘交易
- 风险控制：内置风险控制模块,保障交易安全
- 可视化：提供策略绩效分析和监控图表
